\Zend_Gdata_Gapps_GroupEntry
jump to top

Package: Zend\Gdata\Gapps

Data model class for a Google Apps Group Entry.

Each group entry describes a single group within a Google Apps hosted domain.

To transfer group entries to and from the Google Apps servers, including creating new entries, refer to the Google Apps service class, Zend_Gdata_Gapps.

This class represents in the Google Data protocol.
